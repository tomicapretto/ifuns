<!-- badges: start -->
<!-- badges: end -->

This package contains a Shiny app to interactively visualize univariate functions. To install the package

```r
# install.packages("remotes")
remotes::install_github("tomicapretto/ifuns")
```

And fire the app with 

```r
ifuns::launch()
```
